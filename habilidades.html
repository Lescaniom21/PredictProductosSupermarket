Habilidades


HTML5

HTML5 es el lenguaje estándar para crear páginas web. Permite estructurar contenido mediante elementos semánticos, formularios avanzados y soporte multimedia nativo.

JavaScript

JavaScript es un lenguaje de programación versátil que permite crear interactividad en sitios web, aplicaciones del lado del cliente y servidor, y experiencias web dinámicas.

CSS3

CSS3 es el lenguaje de estilos que permite diseñar y dar formato visual a sitios web, incluyendo animaciones, layouts responsivos y efectos visuales modernos.

Python

Python es un lenguaje de programación de alto nivel conocido por su simplicidad y legibilidad, ideal para desarrollo web, análisis de datos y automatización.

SQL

SQL es el lenguaje estándar para gestionar y manipular bases de datos relacionales, permitiendo consultar, actualizar y administrar datos de manera eficiente.

Bootstrap

Bootstrap es un framework CSS que facilita el desarrollo de sitios web responsivos y modernos con componentes predefinidos y un sistema de grid flexible.

GitHub

GitHub es una plataforma de desarrollo colaborativo que permite alojar y revisar código, gestionar proyectos y construir software junto con millones de desarrolladores.

Git

Git es un sistema de control de versiones distribuido que permite rastrear cambios en el código fuente durante el desarrollo de software.

MySQL

MySQL es un sistema de gestión de bases de datos relacionales de código abierto, conocido por su rendimiento, fiabilidad y facilidad de uso.

SQL Server

Microsoft SQL Server es un sistema de gestión de base de datos relacional, desarrollado por la empresa Microsoft.
